# Game Assets Guide

The game looks for images in this folder (`public/assets`).
Images should be in **PNG** format.
Standard resolution: **1280x720** (16:9 aspect ratio).

## Required Images
Create these files and drop them here:

1.  **bank_entrance.png** (Start Scene: Outside the bank)
2.  **vent_inside.png** (Sneaking: Inside a dusty air vent)
3.  **window_cut.png** (Laser: A round hole cut in glass)
4.  **office.png** (Office: Manager's desk with a donut)
5.  **lobby_guards.png** (Lobby: Guards walking around)
6.  **vault_treasure.png** (Ending: Vault full of diamonds)

## Optional (Fail States)
If not provided, the game just shows the text.
- fail_explosion.png
- fail_fan.png
- fail_glass_floor.png
- fail_donut.png
- fail_shoot.png
